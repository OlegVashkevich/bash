# Bash cкрипты

## Настройки окружения для WSL Debian(12)
```bash
sudo bash install_environment.sh
```
устанавливает на выбор 

- Midnight Commander
- PHP(8.3 + fpm)
- Composer
- Git
- MySQL
- NGINX
- memcached
- opensearch(+jdk)
- opensearch-dashboard
- Munin
- rabbitmq(+erlang)
- minio
- redis
